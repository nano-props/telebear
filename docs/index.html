Features
One terminal session,
everything you need
~

SSH Server

Built-in SSH2 with public-key auth, PTY, shell and exec.

⇋

FRP Tunnel

Auto-generates frpc config, connects to your server, binds a remote port.

>_

Terminal UI

Status, config, logs. Switch views and control services with single keys.

!

Auto Rollback

Tunnel fails? SSH stops automatically. No dangling processes.

%

TOML Config

Six fields. Edit from the TUI or any text editor.

*

Single Binary

Compile to a standalone executable. Ship telebear + frpc.

How it works
Three steps to a
public SSH endpoint
1

Prepare SSH

Validate config, generate host key, write authorized_keys, start SSH2 on localhost.

2

Establish Tunnel

Write temp frpc.toml, spawn frpc, wait for proxy success.

3

Operate

Status, logs, config, stop/restart. Everything in the TUI.

terminalquickstart
