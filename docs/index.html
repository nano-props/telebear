Features

~

SSH Server

Built-in SSH2 with public-key auth, PTY, shell and exec.

⇋

FRP Tunnel

Auto-generates frpc config, connects to your server, binds a remote port.

>_

Terminal UI

Status, config, logs. Switch views and control services with single keys.

!

Auto Rollback

Tunnel fails? SSH stops automatically. No orphan processes left behind.

%

TOML Config

Six fields. Edit from the TUI or any text editor.

*

Single Binary

Compile to a standalone executable. Ship telebear + frpc.

How it works
Three steps to a
public SSH endpoint
1

Prepare SSH

Validate config, generate host key, write authorized_keys, start SSH2.

2

Establish Tunnel

Generate temp frpc.toml, spawn frpc, wait for proxy success.

3

Ready

SSH reachable via FRP. Stop anytime, cleanup is automatic.

terminalquickstart
# install frpc
./setup-frp.sh

# run
bun run start

# or build & run
bun run build
./dist/telebear

# config: ~/.config/telebear/telebear.toml
